# Page Tabs Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement purpose-led page tabs while preserving Orders and Routes v1 behavior.

**Architecture:** Add a flexible shared page-shell component for non-operational pages. Update placeholder tabs to render role-specific sections. Keep existing Orders/Routes modules functionally unchanged.

**Tech Stack:** React 18, React Router 7 route modules, Shopify web components, Node source-level tests.
